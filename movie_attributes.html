Movie Attributes Correlations

Guided data analysis project to show the correlations between various attributes,
like budget, year, rating etc. in the movie industry using Python.

This data analysis project was done under guidance from Alex The Analystâ€™s (Alex Freberg) YouTube channel. The data for this project was obtained from the Movie Industry Dataset in Kaggle.com

First, the pandas, seaborn, numpy, and matplotlib libraries were imported into Python. Then the data was read from the file and a data frame created. Next the data frame was checked for any missing data as well the data type in each column.

The budget and the gross columns were changed from float to integer to get rid of the extra zeroes after decimal point. Then a separate year column was created by extracting the year from the released column after which the data frame was sorted by gross revenue (highest to lowest). Any duplicates were then dropped. Correlation between the following were then visualized through different scatter plots: budget-gross revenue, score-gross revenue.

A correlation matrix was created from the data frame, which was then used to create a heatmap to visualize the correlation between all the movie attributes using seaborn. Then the correlations were unstacked and sorted to show which attributes were the most correlated.

Finally, the highest correlated (greater than 0.5) pairs of attributes were extracted.

According to the Pearson correlation coefficient, votes received by the movies and their budgets had the highest correlation to their gross revenue, 0.712196 and 0.662457, respectively.
