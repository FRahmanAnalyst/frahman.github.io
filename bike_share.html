Bike Share Analysis (Chicago 2019)

Data analysis capstone project for my Google Data Analytics Certificate,
where I analyzed how casual riders and annual members use bike share, and
their demographics, using bike share data from Chicago for the year 2019.

The data for the analysis was obtained from the following link which recorded data from 2004 until the first quarter of 2020. From these, the bike share data for the year 2019 was used in the analysis.

A new column was created for the trip durations (minutes), which was calculated by subtracting the start time from the end time and converting the result into minutes using the SQL commands below.

Download SQL Script

The tables containing data for the four quarters of 2019 were combined using view and union commands in Microsoft SQL Server and then the relevant columns were chosen.

Download SQL Script

Next, the tables were checked for duplicates and missing rows using count command, by checking the number of entries in the table for each quarter, then matching the aggregate number of entries with the combined table.

Download SQL Script

The following were then extracted from the combined data table with SQL commands, to provide an insight on the bike share patterns and the users: number of trips each day of the week, number of trips each month, average trip duration each month, number of trips each week by user type and also the average trip duration each day of the week.

Download SQL Script

SQL commands were used to also extract the following information from the data table: bike share use across age groups, bike share use across age groups in the different user types, number of trips by gender, the number of different user types, the number of different user types by gender, most used bike share start stations, average trip duration in the different start stations, average trip duration by user type and gender, and the number of missing values for gender and birth year across user types.

Download SQL Script

The data was then visualized using Tableau Public

Tableau Public Page

From the visualizations above, we can infer that overall, more bike share trips occur during the working days than weekends, while casual customers tend to use bike share more on the weekends while subscribers use it more on the working days. Subscribers make up the overwhelming majority of the bike trips- more than 75%. More trips occur during the summer months compared to winter months, with July, August and September being the peak months while almost no trips from November to April. There is no gender data from the casual customers, as compared to the subscribers of whom, the overwhelming majority is male. As for the age demographics, 24 to 35 year-olds use bike share the most, with the average age of the subscribers being a little higher than the casual customers.
